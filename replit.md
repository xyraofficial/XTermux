# Replit.md

## Overview

This is an Android project (Termux-related app) migrated to Replit. It uses Gradle for builds and Java 17 (via GraalVM).

## Project Architecture

- **Android App**: Main source code in `app/src/main/java`.
- **Gradle**: Build system with version 8.2.2.
- **CI/CD**: GitHub Actions configured in `.github/workflows/main.yml`.

## Recent Changes

- Migrated project to Replit environment.
- Fixed GitHub Action YML to use Java 17 and modern actions.
- Initialized progress tracker.
